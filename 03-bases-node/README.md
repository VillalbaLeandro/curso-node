````
Iniciar la app con los comandos: node app -b [base] -l [mostrar la tabla en consola] 
                                 -h [hasta que numero mostrar la tabla]
Opciones:
      --help     Muestra ayuda                                                                          [booleano]
      --version  Muestra número de versión                                                              [booleano]
  -b, --base     Es la base de la tabla de multiplicar                                        [número] [requerido]
  -l, --listar   Muestra la tabla en consola                                           [booleano] [defecto: false]
  -h, --hasta    ingrese un numero hasta cuanto quiere mostrar la tabla [número]                     [defecto: 10]

````
